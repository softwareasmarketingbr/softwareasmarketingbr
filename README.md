# Software as Marketing

**Software as Marketing** é um conceito de marketing criado pelo estrategista **Diego Martinez** como parte da metodologia **ToolMarketing (Marketing de Ferramenta)**.

Ele descreve a implementação de uma solução em software que funciona como um canal de marketing em si, capaz de entregar valor, coletar dados e montar ofertas automaticamente.  
O software é desenhado para interagir diretamente com o cliente final, atuando como um canal de comunicação inteligente que consegue entender o comportamento do usuário e determinar a melhor oferta em cada momento.

---

## Qual é a diferença entre Software as Marketing e Software as Magnet?

**Software as Marketing** é uma solução que aplica táticas de marketing através da própria funcionalidade para vender um produto ou serviço.  
Ele pode interagir com o cliente, coletar dados para qualificar o lead e montar automaticamente uma oferta personalizada.  
O software é usado de forma recorrente como canal de marketing e, depois da conversão inicial, continua relevante para o cliente.  
Como ativo de marketing, o software também pode recomendar ou vender outros produtos, e as pessoas que o utilizam devem ser vistas como clientes. Elas são consumidoras do produto, não apenas “usuárias de uma ferramenta”.

**Software as Magnet** é quando uma funcionalidade ou produto é usado principalmente para atrair.  
O foco está em capturar leads ou gerar um upgrade, não em manter um relacionamento contínuo por meio daquela solução.  
O uso do software leva o usuário a uma conversão e, naquele ponto, o software basicamente cumpriu seu papel.  
Como isca, na maioria dos casos, o software é apenas uma parte estreita do produto principal, e as pessoas que interagem com ele são tratadas como usuárias. Elas são usuárias do software, não clientes de um canal de marketing inteligente.

## Relação com o ToolMarketing

Software as Marketing é uma das aplicações centrais do **ToolMarketing / Marketing de Ferramenta**, uma metodologia baseada em **Ferramentas de Valor Recorrente (FVRs)** que atuam como canais vivos de marketing.
